# Moonspire Dream Gate Slice — Design Page

Codex object: moonspire · Source drop: AKALYNTH_MOONSPIRE_DREAM_GATE_SLICE_V1

> "The first slices taught the player to recover truth, reopen roads, and temper power.
> This slice teaches them that possibility is also a terrain." — Akalynth Design Notes

Title: **Moonspire Dream Gate Slice** · Volume I — The Liminal Web · Type: place/route playable slice.

---

## Purpose

Introduce Moonspire, Dreamwalker gameplay, Dream Gates, symbolic puzzles, and the first controlled entry into the Liminal Web.

This slice proves:

- Dreamwalker identity
- dream-state traversal
- symbolic puzzle solving
- emotional evidence
- Dream Sanctum reputation
- controlled dream-gate access
- non-literal dungeon logic
- choice-based reality consequences
- Dream Convergence event seed

This is **not** the full Moonspire city. It is the first playable dream-gate corridor and Liminal Web entry.

**Core premise:** after the earlier memory and route incidents, citizens across High City, Cinderwatch, and the Ember Road report the same dream — a silver web above Moonspire, a door of sleeping water, a moth trapped in a mirror, three arguing versions of themselves, and a voice saying *"One future has already lied."* The Dream Sanctums request assistance; the Codexborn want it recorded; the Trinity Convergence fears imbalance; the Oathbound want to prevent panic. The player must enter the dream safely, interpret symbolic evidence, and determine whether the shared vision is warning, manipulation, or invitation.

**Primary Factions Present:** Dream Sanctums (primary), Codexborn, Trinity Convergence, Oathbound, and the first hinted antagonist — Nightmare Weavers (may be twisting possibility inside the Liminal Web).

## Dream-State Flow (ordered states)

Six-act controlled sequence (not procedural):

1. **Act I — The Shared Dream:** arrive at Moonspire's outer sanctum; establish that dreams can be evidence; Sanctums ask the player to stabilize the dream before civilians become trapped in sleep.
2. **Act II — Reflection Pool Investigation:** recover three symbolic fragments (dream evidence collection).
3. **Act III — Symbolic Puzzle Trial:** dreams follow meaning, not literal logic; interpret symbols rather than match keys to locks.
4. **Act IV — Dream Gate Opening:** open a controlled Dream Gate with faction support.
5. **Act V — Liminal Web Descent:** the first dream dungeon, the Liminal Web — Outer Strand.
6. **Act VI — The Dream That Answered Back:** the Unchosen Self boss and the Integrate / Banish / Bind choice.

**Runtime dream state** tracks: current state (waking / liminal / nightmare pressure), dream stability, emotional residue tags, active symbolic fragments, anchor status.

## Dream Gate System & Dreamwalker Gameplay

**Dream Gates** are controlled thresholds between waking spaces and dream terrain — not normal portals. They open when meaning, memory, and will align.

Requirements: one symbolic fragment · one stabilizing anchor · one interpreted emotional residue · one faction sponsor · safe waking-body protection.

**Gate states:** Dormant → Stirring → Interpreted → Anchored → Open → (hazards) Contaminated / Sealed.

| State | Meaning |
|---|---|
| Dormant | Gate exists but cannot open |
| Stirring | Symbols active but unstable |
| Interpreted | Symbols have been given meaning |
| Anchored | Gate safe enough to enter |
| Open | Liminal Web access active |
| Contaminated | Nightmare pressure has entered the gate |
| Sealed | Gate closed by ritual or emergency order |

**Sponsor effects:** Dream Sanctums → safer traversal · Codexborn → stronger evidence capture · Trinity Convergence → balanced puzzle state · Oathbound → safer civilian extraction · Flamebound → stronger anchor stability · Ashwardens → better hazard detection.

**Failure states:** nightmare leakage · sleeping NPCs trapped · false memory contamination · symbolic lockout · Liminal Web route shift.

**Dreamwalker gameplay** (non-literal play — reads meaning as terrain, not a spellcaster reskin):

- **Lucid Step** — safer movement through unstable dream surfaces; reduces nightmare pressure, reveals temporary safe paths, helps cross Liminal Web strands.
- **Symbol Sense** — reveals possible meanings for dream-symbols; marks associations, improves hints, exposes false interpretations.
- **Emotional Anchor** — stabilizes emotional residue during gate opening; lowers instability, protects sleeping NPCs, improves Dream Sanctum reputation.
- **False Future Resistance** — rejects impossible future projections; reduces disorientation, highlights contradictions between Chronicle memory and false possibility.

## Symbolic Puzzles

Dream logic is not literal. A locked door may not require a key — it may require naming the fear that made it locked.

**Puzzle pattern** — each puzzle asks: 1) What is seen? 2) What is felt? 3) What is avoided? The answer is the *relationship* between symbols, not the obvious object.

| Symbol | Possible Meaning |
|---|---|
| Door | fear, boundary, permission, refusal |
| Mirror | identity, deception, self-recognition |
| Moth | recorded truth, witness, fragile memory |
| Water | hidden memory, emotion, unconscious knowledge |
| Web | connected possibility, fate, influence |
| Flame | will, transformation, danger, purpose |
| Crown | authority, burden, inherited role |
| Road | choice, direction, return, exile |

**Example:** seeing a locked door, a mirror behind it, and water underfoot — the literal answer is "find a key"; the symbolic answer is that the player is avoiding self-recognition of a hidden memory. Correct action: look into the water reflection, name the fear, open the door without a key. Design rule: a symbolic puzzle must be **readable** — the player should feel clever, not tricked.

**Three recovered fragments** (Act II evidence):

| Fragment | Location | Gameplay | Threat | Used For |
|---|---|---|---|---|
| Silver Thread | Reflection Pools | align water reflections to reveal a hidden dream-symbol | mirrored self-doubt echoes | opening the first Dream Gate lock |
| Sleeping Moth Scale | Memory Garden Gate | protect a dream-touched Witness Moth projection | nightmare moth larvae | distinguishing memory from possibility |
| Broken Future Shard | Silent Observatory Terrace | inspect three possible futures, identify the one with impossible evidence | unstable possibility bursts | the symbolic puzzle trial |

(Emotional Residue, recovered in the Dream Sanctum Antechamber by interpreting the tone shared dreamers left behind, determines gate stability and faction interpretation.)

## Key Locations

Region: **Moonspire Outer Sanctum** (not full Moonspire — the first controlled dream-access slice). Eight location nodes:

- **Moonspire Pilgrim Road** — approach route; introduces dream-touched travelers and shared-dream rumors.
- **Reflection Pools** — primary symbolic investigation area; water reflects emotional truth, not physical appearance.
- **Dream Sanctum Antechamber** — main faction hub; guides, sleeping citizens, ritual preparations.
- **Silent Observatory Terrace** — open-air platform where possible futures appear as star alignments; Broken Future Shard recovery.
- **Memory Garden Gate** — boundary between remembered history and emotional residue; Sleeping Moth Scale recovery.
- **Dream Gate Hall** — controlled chamber where the first Dream Gate opens.
- **Liminal Web Threshold** — transition point from waking space into dream terrain.
- **Webbed Mirror Chamber** — boss arena for the Unchosen Self.

**Dungeon — The Liminal Web — Outer Strand:** not a physical ruin but a structured possibility-space where symbolic decisions alter paths. Target 30–45 min; solo-capable with optional 3-player tuning; themes of dreams, possibility, identity, symbolic truth, fear, alternate selves.

| Room | Mechanic | Purpose |
|---|---|---|
| 1 — The Sleeping Door | door opens only when the player chooses what it *represents*, not what it is | teach symbolic interaction |
| 2 — Hall of Three Selves | meet three dream-selves: the self that remembers, fears, and chooses | introduce identity as gameplay |
| 3 — Threaded Crossing | cross glowing web strands in order of emotional logic; wrong order creates nightmare pressure | teach non-spatial pathfinding |
| 4 — The Mirror That Lies | reflection shows a false future; compare dream-shard evidence against the mirror vision | teach dream verification |
| 5 — The Cocooned Gate | free or seal a trapped dream-symbol before the final encounter | prepare the final choice |

## Boss — The Unchosen Self

A living possible version of the player that was never allowed to exist. It speaks with the player's voice, remembers choices never made, and tries to replace the waking self. It is not purely evil — it is resentment given form.

**Mechanics:**

1. **Alternate Memory** — summons memories of events the player never experienced; the player must identify which belongs to the real Chronicle record.
2. **Fear Door** — the arena creates symbolic doors; only the door representing the player's true unresolved fear leads to safety.
3. **Web Pull** — drags the player toward false future strands; the Dreamwalker path gives stronger resistance.
4. **Mirror Claim** — the boss claims one faction would prefer the alternate self; the player must reject or accept using gathered evidence.
5. **Final Awakening** — at low health, the player chooses how to handle the Unchosen Self (Integrate / Banish / Bind), determining the slice resolution.

**Integrate / Banish / Bind:**

| Choice | Meaning | Supported By | Consequences | Reward · Theme |
|---|---|---|---|---|
| Integrate | accept the Unchosen Self as a symbolic part of the player | Dream Sanctums; some Trinity Convergence mediators | Dreamwalker insight improves, future symbolic puzzles clearer, nightmare resistance improves, emotional risk remains | Unchosen Insight · *Understanding transforms fear* |
| Banish | reject and force it out of the dream | Oathbound; some civilian leaders | immediate safety improves, fewer dream leaks in Moonspire, Dream Sanctum concern increases, future Nightmare Weaver thread strengthens | Waking Seal · *Some doors should stay closed* |
| Bind | seal it inside a controlled dream-vessel for study | Codexborn; cautious Dream Sanctum scholars | dream evidence preserved, research path unlocks, ethical concern rises, Liminal Web reacts later | Dream Vessel · *Knowledge can become custody* |

## Release Gates / Success Criteria

**Release gates:**

| Gate | Requirement |
|---|---|
| Dreamwalker Gate | Dreamwalker identity changes interaction with dream systems |
| Symbol Gate | player solves at least one puzzle through symbolic interpretation |
| Dream Gate Gate | player can open and enter a controlled Dream Gate |
| Liminal Gate | Liminal Web changes space based on meaning or emotion |
| Boss Gate | Unchosen Self mechanics reinforce identity and possibility |
| Choice Gate | Integrate / Banish / Bind changes faction and future event state |
| Chronicle Gate | Dream Gate opening is recorded |
| Expansion Gate | deeper Moonspire and Dreamweaver remain teased but locked |

**Success criteria** — the slice succeeds if the player can say: *I entered Moonspire for a reason · I learned that dreams can be evidence · I solved a puzzle by meaning, not object matching · I opened a Dream Gate · I entered the Liminal Web · I faced a possible version of myself · I made a meaningful choice about identity · The world recorded the dream event.* If the player only says *"I went to a purple dungeon and killed dream monsters,"* the slice failed.

**Unlocked at slice end:** Moonspire Outer Sanctum access · Reflection Pool repeat puzzles · Dream Gate Hall access · Liminal Web Outer Strand repeat access · first Dream Silk crafting recipe · Dream Sanctum reputation track · Dream Convergence event seed. **Not yet unlocked:** full Moonspire city · deeper Liminal Web · Dreamweaver encounter · Nightmare Bloom event · Dream Queen legacy chain.
